h1.Deploy
1. ```npm install```
2. Create a ```server/config/providers.json``` file
3. ```webpack```
4. ```npm start```

h1.Development Server
1. Run ```npm run dev```
2. Access ```http://localhost:3000/webpack-dev-server/public```
